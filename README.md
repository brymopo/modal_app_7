# README

A simple project for practice creating a modal on rails 7 with Hotwire, TurboJs and Stimulus Js.

Things you may want to cover:

* Ruby 3.0.2

* Rails 7.0.1

* Node v16.13.2

## INSTRUCTIONS

1. Clone repository

2. `cd ./modal_app_7`

3. `bundle install`

4. `bin/dev` to start the server
